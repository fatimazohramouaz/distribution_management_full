Distribution Management - Full for Odoo 17
------------------------------------------
This expanded module includes:
- Products and categories
- Clients (customers) with debt computation and payment history
- Suppliers and basic Purchase Order model
- Integration placeholders for stock moves (connects to stock module)
- Sales Orders (with automatic stock reservation placeholder)
- Tours with delivery tracking (delivered qty)
- Security groups: Distribution Manager, Sales Agent, Accountant
- Simple QWeb report template placeholders for invoices and tour reports
- Data sequences for orders
- project12.pdf (user-provided plan) included in root of module

Notes:
- This module is a scaffold and includes business logic placeholders where deeper Odoo integration is required.
- After installing, you may need to customize workflows, accounting journals, and additional reports.
